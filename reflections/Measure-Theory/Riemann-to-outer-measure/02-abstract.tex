\begin{abstract}
    This project presents a systematic mathematical study of the Riemann Integral and its limitations in measuring a set, which eventually leads to the Outer measure and its properties. The Outer measure creates the building block before setting the axiomatic, abstract and more advanced tools for the Measure Theory. Thus, this project is crucial for developing the true intuitions prior to the rigorous measure theory, a cornerstone in modern mathematics. It contains key theorems, definitions, and proofs, reflecting my own understanding and originality. Although most of the mathematical sections in this project are truly inspired from the graduate level textbook \textit{Measure, Integration \& Real Analysis} by \textit{Sheldon Axler}, this project only accumulates my own concretized versions of this reference. This project ends with the key exercises from the same book of Axler, with the detailed solution distilled with emphasis and intuition. This exercise section developed through independent study and problem solving. The transition from Riemann integration to outer measure is explored as a foundational step 
toward abstract measure theory
\end{abstract}
